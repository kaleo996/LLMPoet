# NekooBasho Slides (Slidev)

This directory contains the Slidev deck for the NekooBasho project.

## Local development

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Open the URL shown by Slidev (usually `http://localhost:3030`)

## Build

- Build static files: `npm run build`
- Output directory: `dist/`

## Deploy to `gh-deploy`

- Run: `npm run deploy:gh`
- This command builds the deck and publishes `dist/` to the `gh-deploy` branch.
